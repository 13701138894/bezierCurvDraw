# bezierCurvDraw
基于贝塞尔曲线拟合算法实现绘制平滑曲线

## 算法
 *  JavaScript implementation of
 *  Algorithm for Automatically Fitting Digitized Curves
 *  by Philip J. Schneider
 *  "Graphics Gems", Academic Press, 1990

## 实现
1. 基于拟合算法生成对应的贝塞尔曲线点
2. 基于生成的贝塞尔曲线点生成对应的绘制点
3. 基于绘制点生成对应的线条
